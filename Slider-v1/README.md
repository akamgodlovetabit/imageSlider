
### **Adjust Animation Styles**  
Modify CSS variables in **style.css**:  
```css
.watermark {
    color: rgba(0, 0, 0, 0.1); /* Adjust opacity */
    font-size: 5rem; /* Change size */
    transform: rotate(-15deg); /* Tilt angle */
}
```
